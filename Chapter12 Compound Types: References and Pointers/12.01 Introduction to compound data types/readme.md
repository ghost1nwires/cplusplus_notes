## Intro to compound data types

C++ sipports second data types which is : **compound data types** (***sometimes called composite data types*** ) are types that are defined in terms of other existing data types.

C++ supports the following compound types:

- Functions
- C-style Arrays
- Pointer types:
    - Pointer to object
    - Pointer to function
- Pointer to member types:
    - Pointer to data member
    - Pointer to member function
- Reference types:
    - L-value references
    - R-value references
- Enumerated types:
    - Unscoped enumerations
    - Scoped enumerations
- Class types:
    - Structs
    - Classes
    - Unions
